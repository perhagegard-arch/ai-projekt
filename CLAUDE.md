# Projektregler

## 1. Kommentarer på svenska
Alla kommentarer i koden ska skrivas på svenska.

## 2. Mörkt tema som standard
Alla webbsidor som byggs i det här projektet ska ha ett mörkt och stilrent utseende (dark mode) som standard, om inget annat begärs.

Exempel på basinställningar för dark mode:
```css
body {
  background-color: #121212;
  color: #e0e0e0;
  font-family: sans-serif;
}
```
